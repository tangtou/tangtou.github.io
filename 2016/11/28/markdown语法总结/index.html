搭建好博客后需要在终端里进入博客文件，然后 hexo new example 出一个文件来才能发到博客上，格式是.md，在Mac下需要通过Mou打开，然后编写。此文总结markdown的语法。

一、通过在文字下方添加“=”和“-”，他们分别表示一级标题和二级标题。

二、在文字开头加上 “#”，通过“#”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）

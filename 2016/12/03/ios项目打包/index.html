ios项目打包

ios项目打包

react-native 发布流程:
准备工作:
1、苹果开发者帐号一枚
https://developer.apple.com/
2、蒲公英应用测试号一枚
https://www.pgyer.com/

开发者中心
1、打开网址，点击右上方的Account进行登陆
2、点击 Certificates,IDs & Profiles
3、找到Devices 点击选项ALL点击右上角+号进行添加设备UDID 之后点击Continue和Register如下图：


添加UDID
4、Identifiers -> App IDs -> 添加APP ID也就是bounld ID
点击Continue和Register如下图：

Bundle Id 就是此项里面的内容

5、最下面 Provisioning Profiles -> Distribution->添加描述文件－>点击添加->Ad Hoc->继续->选中之前添加的App ID->继续－>选中证书->继续－>选择设备->继续->添加描述文件名称－进入到下载描述文件页，点击下载



下载权限可以全选，也可以只选择部分人员。


6、找到你下载好的有效证书和下载好的描述文件双击打开

7、找到要打包的文件夹，文件用Xcode打开，点击Build Settings –>All和Combined –>Code Signing –> Code Signing Identity和Provisioning Profile 选择正确的证书和描述文件


8、然后将Xcode左上方的模拟机选成默认的


9、选择Product–>Archive

10、打包成功 点击Export–>Save for Ad Hoc Deployment






11、文件夹里会出现后缀为.Ipa的文件证明打包完成

12、点击此网站https://www.pgyer.com/上传即可

13、扫描二维码获取UDID
